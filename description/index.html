公司描述

Aug 24, 2019

华鼎环保科技有限公司（成都分公司）成立于2010年9月，成都市城管办、工商局批准，注册成立的一家大型成都专业清洁公司，属成都清洁协会会员单位，在国内清洁行业通过英国BSI第三方ISO9001国际质量管理体系认证，拥有规范的质量管理体系，确保了服务质量的稳定和提高，资质齐全，清洁设备齐全，美莱清洁力争为客户和用户提供一流的专业清洁服务，主要服务项目：物业保洁、日常保洁、开荒清洁、外墙清洗、地毯清洗、水池清洗，地板清洗、石材养护、地板打蜡等物业清洁服务。
